Response Drone for Remote Areas



Project Description



This project is designed to simulate a drone-based emergency response system for remote areas. The system accepts emergency requests from villages, assigns a priority level, allocates a drone, estimates delivery time, and stores request details in a file.



Features



\- Emergency request registration

\- Priority classification (High, Medium, Low)

\- Automatic drone assignment

\- Estimated delivery time calculation

\- Request ID generation

\- Request storage in text files



Technologies Used



\- Python

\- File Handling

\- Conditional Statements

\- Loops
